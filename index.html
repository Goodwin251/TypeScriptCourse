TaskManager

Project description

edited September 21, 2024

There is simple project, that let you write and mark tasks in open browser. It used TypeScript to write source, than it compile code in JavaScript. Page should be hosted via http-server, eslint using for tracking purpose. For web-site I used bootstrap template.
